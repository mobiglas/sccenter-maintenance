On StarCitizen.center you will find a collection of Star Citizen companion apps, similar to MobiGlas ones but with additional features.

The Route Planner is the first app we built with following main features:

  • Fastest path route algorithm with variables like ship (jumppoint) size, threat level (of each system and place) and affiliation.
  • Custom map layout - you can manually arrange system positions on the map by using drag&drop (share/export soon).
  • Approximate calculation of travel time, distance and fuel usage (work in progress, and waiting for StarCitizen 3.0) for each segment of the route.
  • Comprehensive (official) list of places (planets, asteroid belts, stars etc) with custom properties like "mining". Easy to find a new adventure or business location.
  • Browser persistence between sessions (route places and preferences, favorites, system positions on the map)
  • Tablet friendly - designed to work on tablets, as a companion app for Star Citizen.
  • Share (not implemented yet) your routes and system layout (positions on map).
